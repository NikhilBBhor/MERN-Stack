Select backend folder --> cd backend
Initialise npm --> npm init -y
Install express(specific version) --> npm install express@4.18.2
Install nodemon --> npm install nodemon -D (D means as dev dependancies)
Install mongoose --> npm install mongoose
Install dotenv --> npm i dotenv
Installation for ratelimiting --> npm i @upstash/ratelimiting@2.0.5 @upstash/redis@1.34.9
Install cors package --> npm i cors@2.8.5

Select frontend folder --> cd frontend
Install vite --> npm create vite@latest . --> y --> React --> JavaScript
Install npm --> npm install
Run the app --> npm run dev
Install react router --> npm i react-router 
Install react hot toast --> npm i react-hot-toast
Install ES7 extension --> rafce (creates boiler plate)
Install tailwind css --> npm install -D tailwindcss@3 postcss autoprefixer
Initialise tailwindcss --> npx tailwindcss init -p
Install tailwind css intellisence extension --> autocomplete tailwindcss classes
Install daisy ui --> npm i daisyui@4.12.24 -D
Install lucide react icon library --> npm i lucide-react
Install axios package --> npm i axios
